Для связного неориентированного графа, заданного списком рёбер, определить,
является ли он полуэйлеровым (то есть содержит ли эйлеров путь). Если да,
вывести вершины пути, иначе - вывести NO.

Пример 1:
  input.txt:
    7
    0 1
    0 2
    1 0
    1 2
    1 3
    2 3
    3 1
  output.txt:
    NO

Пример 2:
input.txt:
  8
  0 1
  0 2
  1 0
  1 2
  1 3
  2 3
  3 1
  3 2
output.txt:
  0 1 0 2 1 3 2 3 1
