Вход:
строка 1:
слово
со 2-й строки:
словарь - каждое слово с новой строки

Выход:
YES - если слово в словаре, NO - в противном случае

Для решения задачи использовать массив указателей на списки слов.
Для определения сегмента словаря использовать функцию, определяющую для слова,
целое число - средний ASCII-код символов слова.

Пример 1
Вход:

ФИТ
НГУ
программирование
информатика
ФИТ
сообщение

Выход:
YES


Пример 2
Вход:

ФИТ
НГУ
программирование
информатика
сообщение


Выход:
NO
